Tie-_breaker_app
AI-powered Tie Breaker application that helps users make quick decisions between multiple choices using Google Gemini AI.
 📖 Overview:

AI Tie Breaker is designed to solve everyday decision-making problems by analyzing user inputs and providing a smart, logical, and AI-generated recommendation. It acts like a digital decision assistant that helps users eliminate confusion when choosing between options.
 Features:

* Compare two or more options
* AI-generated decision with reasoning
* Simple and fast user interface
* Context-aware suggestions
* Useful for daily life decisions (food, study, work, etc.)

Technologies Used:

* Google AI Studio
* Gemini API
* Python
* Prompt Engineering
* Git & GitHub
